# molegame (HTML)

## 프로젝트 소개  
간단한 슈팅 게임 형태의 웹게임으로, 제한 시간 내 무작위로 나타나는 타겟을 클릭해 점수를 획득하는 것을 목표로 제작했습니다.

## 주요 기능  
- 무작위 타겟 등장 및 점수 집계  
- 게임 시간 제한 및 카운트다운 기능  
- 마우스 커서 효과 및 UI 전환

## 사용 기술  
HTML5, CSS3, JavaScript
